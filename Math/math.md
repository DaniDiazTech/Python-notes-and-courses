# Geometria analitica

# Linea recta:

# Lugar geometrico:
  Conjunto de puntos del plano cartesiano que  cumplen con una caracteristica
  geometrica común.

# Distancia entre dos puntos:

~~~
  A(x1, y2)
  B(x2 , x2)

  d(A, B) = squareroot( (x2 - x1) ***** 2 + (y2 - y1)** 2 )

~~~
  
# Punto medio

~~~                                      
  ( x1 + x2 / 2 ,  y1 + y2 / 2)                                                                 
~~~

# Pendiente de la recta = m :

~~~
   P( x1 , y1)
   Q(x2, y2)

   m = (y2 - y1 / x2 - x1 )
~~~
    * pendiente tambien puede ser tan x:


~~~
      m = tan x
      tan x = (y2 - y1 / x2 - x1 )

~~~

# Ecuación de la recta

~~~
  # Ecuacion canonica
  Q(x1 , y1) => y = mx + b
  
  y - y1 = m(x - x1)

~~~

# Ecuación general


~~~
   
  Ax + By + c = 0
  # siempre y cuando
  A,B,C  E/  R|
~~~
